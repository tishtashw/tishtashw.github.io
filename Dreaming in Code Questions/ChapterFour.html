  1. What do “front ends” and “back ends” mean to software developers?
    • The front end is what the users interact with, allowing for the input and output of information while the back end is what communicates all of this to the computer. The equivalent of C3PO and R2D2.
  2. What did the Lego Hypothesis refer to?
    • Reusable code that can be put together like legos to create a new program faster
  3. Give one reason why the Lego Hypothesis seems to not work so well.
    • Becuase not all code is made the same and with so many different platforms and operating systems many coders feel they can DIY the code themselves in less time than it would take to find prewritten code and adapt it to their needs
